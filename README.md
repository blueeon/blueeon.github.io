# blog

## 约定

### 目录
* spring：计划/起点/希望/
* summer：奋斗/过程/兴奋/热烈
* autumn：收获/总结/幸福
* winter：总结/终点/低落

### 命令
* 文档路径：`~/CloudStation/应用/hugo/...`
* 创建文章：`hugo new [spring|...] title.md`
* 本地预览：`hugo server --buildDrafts`
* 发布：
    1. `hugo`
    2. `git push`
