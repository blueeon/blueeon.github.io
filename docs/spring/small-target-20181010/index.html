一点阶段小目标

2018年10月10日 · 188字 · 1分钟

武汉的工作算是告一段落。 搭建团队的工作基本快完成了，几天算了一下，已经90人了，还有三个月，完成几乎是一定的。但是接下来马上有了新的麻烦：

  1. 如何展现产出？
  2. 多次one on one反馈出来的流程和开发模式问题

所以接下来到年底，要集中精力做两件事情：

  1. 找到几个标杆的本土项目，
  2. 尝试从武汉推动开发模式乃至落后的技术体系的改革

就此立个flag吧，还要为明年春季的评级做一个中期的准备，升它一级。
